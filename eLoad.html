Problems Faced

  1. Manual control
  2. Time consuming
  3. No remote control
  4. Accuracy (current input) and varying current wrt time (hall sensor)
  5. No data saving
  6. Turns off by pressing button (above remote operation)

Solution

  1. Remotely controlled E load operation
  2. Start stop and pause
  3. Status
  4. Voltage cut-off
  5. Time prediction of discharge
  6. Real-time monitoring of E load (no need of manpower to monitor physically for entire duration)
  7. Data saving of E load in .csv (v,i,p, timestamp)
  8. Feature to input constant/varying/cyclic/custom program based current wrt time
  9. User specific program
  10. Accurate value of voltage and current(number of decimals)
  11. Offline data logging without internet
  12. If offline, time for offline
  13. Accurate time based program
